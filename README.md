# 팜한농 청양대리점 마케팅 홈페이지

충남 청양의 팜한농 대리점 마케팅 홈페이지입니다. 지역 특화 제품 큐레이션 및 기술력 비교를 통해 농업인에게 최적화된 정보를 제공합니다.

## 🔗 Live Demo
**[팜한농 청양대리점 홈페이지 바로가기](https://seungmin-park-psm1757.github.io/farm-marketing-for-jongmin/)**

## 주요 기능
- **지역 특화 제품 추천**: 고추, 토마토, 멜론 등 청양의 주요 작물별 맞춤 비료 추천
- **기술력 비교 차트**: 팜한농의 완효성/광분해 코팅 기술(에코뮬라)의 경제성 시각화
- **간편 상담**: 이종민 팀장 직통 전화 및 카카오톡 상담 연결

## 기술 스택
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS, Framer Motion
- **Visualization**: Chart.js
- **Deployment**: GitHub Pages (GitHub Actions)

## 로컬 실행 방법
```bash
npm install
npm run dev
```
http://localhost:3000 에서 확인하실 수 있습니다.
